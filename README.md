## Need

- PHP 7.4+
- Classic/Normal Web host

## 安裝

1. Clone to somewhere
2. make `data/` (If needed)：
   ```bash
   mkdir -p data/brand data/orders data/translations
   chmod 755 data data/brand data/orders data/translations
   ```
3. Start `admin.php`，setup a passcode。
